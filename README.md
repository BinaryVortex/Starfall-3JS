# Starfall-3JS

An interactive starfall animation built using Three.js and WebGL shaders.

## Description

This project creates a mesmerizing starfall effect with multiple layers of particles that fall and rotate in a cosmic scene. The animation uses custom GLSL shaders to generate dynamic, colorful starfields with depth and motion.

## Features

- Multi-layered particle system for depth
- Real-time shader-based rendering
- Mouse interaction for dynamic effects
- Screen recording capability (press 'D' to toggle)
- Responsive design that adapts to window size
- Smooth animations with rotation and scaling

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/BinaryVortex/Starfall-3JS.git
   cd Starfall-3JS
   ```

2. Open `index.html` in your web browser.

That's it! The project uses CDN-hosted libraries, so no additional setup is required.

## Controls

- **Mouse Movement**: Interact with the animation
- **Mouse Click**: Additional effects
- **Press 'D'**: Toggle screen recording (saves as WebM file)

## Technologies Used

- **Three.js**: 3D graphics library
- **WebGL/GLSL**: Shaders for rendering
- **CCapture.js**: Screen recording functionality

## Browser Compatibility

Requires a modern browser with WebGL support.
